About DevSpace

DevSpace is an experimental developer namespace project by Anvesh Raman.

Our Mission

DevSpace was created to provide developers with clean, memorable subdomains for their projects, portfolios, and documentation. We believe every developer deserves a professional web presence without the barrier of domain costs.

What We Offer

  • Free subdomains in the format username.devspace.qzz.io
  • Support for GitHub Pages and Vercel hosting
  • Manual review to ensure quality
  • Simple JSON-based registration process
  • Community-driven registry on GitHub

How It Started

DevSpace began as a simple idea: make it easy for developers to get a professional subdomain without the hassle and cost of purchasing a domain. By leveraging GitHub for the registry, we created a transparent, community-driven system where anyone can contribute and review changes.

The project is maintained by volunteers who believe in supporting the developer community. All domain registrations are processed through Pull Requests, ensuring transparency and giving the community a voice in what gets approved.

Supported Platforms

We currently support two of the most popular static hosting platforms:

GitHub Pages

Netlify

Both platforms offer excellent performance, SSL certificates, and global CDN distribution — perfect for developer portfolios and project sites.

Open Source

DevSpace is built on open source principles. The entire registry is hosted on GitHub, making it transparent and community-driven. Anyone can:

  • View all registered domains
  • Submit new domain requests via Pull Request
  • Report issues or suggest improvements
  • Contribute to the project
