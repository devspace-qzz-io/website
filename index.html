How It Works

DevSpace provides free subdomains for developers who want a clean, memorable URL for their projects. All requests are manually reviewed to ensure quality.

Supported formats: username.devspace.qzz.io

  1. Fork the registry repository on GitHub
  2. Add your subdomain configuration as a JSON file
  3. Open a Pull Request for review
  4. Once approved, your subdomain will be active

Requirements

  • Your site must be hosted on GitHub Pages or Vercel
  • Content must be developer-related (portfolio, project, documentation)
  • You must provide a live site or public repository
  • Custom domain must be configured before final approval
